# Changelog

All notable changes to this Terraform module will be documented in this file.

## [1.0.0] - VPC Endpoint Resources

- Added map-based VPC endpoint creation.
- Added Gateway endpoint support with route table associations.
- Added Interface endpoint support with subnet and security group associations.
- Added enterprise tag validation.
- Added endpoint IDs, ARNs, service names, DNS entries, and route table outputs.

## [0.1.0] - Initial Structure

- Added enterprise Terraform module repository structure.
